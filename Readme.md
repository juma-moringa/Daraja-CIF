## Description
Daraja CIF is a Django website application that showcase all about daraja as a whole.

    - Python3.9.1
    - Postgres
    - python virtual environment (virtual:venv).

### Technologies used
    - Python 3.9.1
    - HTML5
    - Django 3.2.5
    - Bootstrap 5
    - Postgresql
    - GIT
    - Material Design
